## Installation

Install dependencies. We need to manually use version 0.11.1 of Glide temporarily because 0.12 introduced a bug in recursive dependencies.

Install Glide.
```sh
make install-glide
```

Get dependencies.
```sh
make deps
```

Build.
```sh
make
```

## Testing

```
make test
```
